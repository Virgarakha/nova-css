    novacss adalah framework css yang dibuat untuk mempermudah styling pada website portofolio, e - commerce dll. framework ini dibuat pada tahun 2024

    Quick Start

    Get started by incluiding novacss production ready css and via npm

    Gunakan CDN

    https://cdn.jsdelivr.net/gh/Virgarakha/nova-framework/dist/css/main.min.css

    Gunakan NPM

    $ npm init -y

    $ npm install @rakaror777/nova-css

    Lets go using this framework!
